Welcome to City4Age Rest API

Here you have a list of available commands to use:

  • login: Obtain an identification user token.
  • logout: Remove the actual user token from the system.
  • add_action: Adds new Action into database.
  • add_activity: Adds new Executed Activity into database related with some actions.
  • add_new_activity: Adds new Activity into the Activity codebook.
  • add_measure: Adds a new Measure into database.
  • add_eam: Adds information about EAM's in the API related to an activity.
  • add_care_receiver: Allows to a Pilot add a new user 'care_receiver' in the API.
  • add_new_user: Adds a new registered user in the system (Administrator only).
  • add_new_activity: Adds a new activity into the activity codebook (Administrator only).
  • search[beta]: Makes a search based on a certain criteria.
  • get_my_info: Returns user information about the actual client.
  • add_factor: Manually adds the care recipient factor. Available values are 'frail', 'non-frail', 'pre-frail'.
